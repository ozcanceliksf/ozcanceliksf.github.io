# ozcanceliksf.github.io 

Personal portfolio website — built from scratch with HTML, CSS, and vanilla JavaScript.

🌐 **Live:** [ozcanceliksf.github.io](https://ozcanceliksf.github.io)

---

## About

This is my personal portfolio site showcasing my IT Security studies, technical projects, CTF progress, work experience, and background. Designed with a dark hacker aesthetic using neon colors and monospace typography.

---

## Features

- **Typing effect** hero section
- **Scroll reveal** animations
- **Animated skill bars**
- **Live GitHub repos** — fetched automatically via GitHub API
- **TryHackMe badge** — updates automatically as I progress
- **PicoCTF challenges** — documented solved challenges
- **Responsive design** — works on mobile and desktop
- **No frameworks** — pure HTML, CSS, JavaScript

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (Intersection Observer, Fetch API)
- Google Fonts (Share Tech Mono, Rajdhani, Inter)
- GitHub API (live repo cards)
- GitHub README Stats (stats cards)
- Streak Stats (contribution streak)
